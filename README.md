# 🐾 양구, 발걸음 가볍개 (양구 애견동반 지도 판별기 MVP)

> **양구군 내 반려동물 동반 장소와 방문 전 확인사항을 30초 만에 확인하고 동적 지도로 탐색하는 No-AD, Pure Data 기반 정적 웹 유틸리티**

본 서비스는 수도권 등에서 양구를 방문하려는 반려견 보호자가 반려견의 크기, 여행 목적(카테고리), 검색어 및 정렬 기준을 선택하면, 양구군 내의 반려동물 동반 가능 후보 장소들을 **오픈소스 동적 지도(Leaflet.js)** 위에 핀으로 즉시 띄우고, 각 장소별 맞춤형 방문 확인 질문 복사 및 정보 공유 기능을 제공하는 공익형 웹 가이드북입니다.

---

## 💻 서비스 화면 구성 (2열 레이아웃)
1. **Header**: 🐾 꼬리 흔들기 애니메이션 로고 및 서비스 타이틀
2. **Sidebar (좌측 32%)**:
   - 실시간 텍스트 검색창 (장소명, 주소, 꿀팁 키워드 매칭)
   - 카테고리 탭 필터 (전체, 숙소, 식당/카페, 관광지, 산책로)
   - 반려견 크기 조건 칩 (소형견, 중형견, 대형견 다중 And 연산 필터)
   - 결과 정렬 드롭다운 (신뢰도순, 공식정보순, 지도좌표순, 전화유무순)
   - 검색 개수 및 결과 카드 리스트 (클릭 시 지도 이동 및 상세 정보 오픈)
3. **Map Area (우측 68%)**:
   - OpenStreetMap 기반의 Leaflet.js 반응형 고해상도 지도
   - 카테고리별 맞춤형 커스텀 DivIcon 핀 마커 및 활성화 상태 줌 인 모션
   - 모바일 환경 전용 플로팅 목록-지도 뷰 전환(Toggle) 버튼
4. **Detail Panel (우측 슬라이드-인)**:
   - 장소별 실시간 테마 사진 매칭 및 데이터 출처, 최종 갱신일 표시
   - 4대 주요 포털 바로가기 (네이버 지도, 카카오맵, 네이버 검색, 구글 검색) 자동 연동
   - 전화 걸기 및 공식 정보 홈페이지 링크 원클릭 연결
   - 방문 전 업주/관리자 필수 문의 질문 템플릿 아코디언 및 질문 복사
   - Web Share API 모바일 네이티브 공유 및 클립보드 링크 복사 폴백

## 🚀 실행 및 로컬 테스트 방법

### 1. 로컬 정적 서버 구동 (강력 권장)
브라우저의 비동기 `fetch()` API에 대한 로컬 파일 보안 정책(CORS)을 준수하기 위해 로컬 웹 서버 구동을 강력하게 권장합니다.

```bash
# Python이 설치된 경우 (포트 8000)
python -m http.server 8000

# Node.js가 설치된 경우 ( serve 모듈 즉시 실행 )
npx serve .
```
구동 후 인터넷 브라우저로 `http://localhost:8000` (또는 터미널에 표시되는 포트)에 접속합니다.

### 2. 브라우저로 파일 직접 열기
인터넷이 연결된 환경에서 브라우저 종류(Safari 등 일부 브라우저)에 따라 `index.html` 파일을 더블클릭하여 바로 구동해볼 수도 있습니다. (단, 크롬 및 엣지 브라우저는 보안 정책 상 데이터 조회가 차단될 수 있으므로 정적 서버 구동을 권장합니다.)

## 🌐 무료 배포 방법 (GitHub Pages)
1. GitHub 개인 계정에 새 Public 리포지토리를 생성합니다.
2. 로컬 작업 공간의 모든 파일(폴더 구조 유지)을 업로드(Commit & Push)합니다.
3. 리포지토리의 `Settings` -> `Pages`로 이동합니다.
4. Build and deployment의 Source를 `Deploy from a branch`로 설정하고 Branch를 `main` (또는 master)의 `/root`로 지정 후 `Save` 버튼을 클릭합니다.
5. 1분 후 `https://<유저네임>.github.io/<리포지토리명>/` 주소로 전 세계에 즉각 무료 배포됩니다.
